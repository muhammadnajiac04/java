# MCA Java Programming Lab ☕

This repository contains Java programs and assignments completed during the **2nd Semester of the Master of Computer Applications (MCA)** program at **APJ Abdul Kalam Technological University (KTU)**.

## 📌 Project Overview
The collection covers fundamental to advanced Java concepts, focusing on Object-Oriented Programming (OOP) principles, data structures, and various practical implementations required by the university syllabus.

## 📂 Repository Structure
The programs are organized into folders corresponding to specific lab cycles or experiment numbers:

| Folder | Description |
| :--- | :--- |
| `1 - 6` | Core Java basics (Classes, Objects, Constructors, Strings). |
| `8 - 12` | Advanced OOP (Inheritance, Polymorphism, Interfaces, Packages). |
| `13 - 18` | Exception Handling, Multithreading, and Collection Frameworks. |
| `*.pdf` | Reference lab records and experiment documentation. |

## 🛠️ Key Topics Covered
* **Encapsulation & Inheritance:** Implementing hierarchical structures.
* **Polymorphism:** Method overloading and overriding.
* **Interface & Abstraction:** Decoupling code using abstract classes and interfaces.
* **Exception Handling:** Robust error management using `try-catch-finally`.
* **Multithreading:** Concurrent execution and thread synchronization.
* **File I/O:** Reading from and writing to files.

## 🚀 Getting Started

### Prerequisites
* **JDK (Java Development Kit) 11 or higher**
* A text editor (VS Code, Sublime Text) or an IDE (IntelliJ IDEA, Eclipse).
